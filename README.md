# jcXue001.github.io (AstroZen-style)

This is a minimal personal website built with [Astro](https://astro.build/) in an AstroZen-like style.

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

This repo includes a GitHub Actions workflow at `.github/workflows/deploy.yml` that builds and deploys `dist/` to GitHub Pages.
